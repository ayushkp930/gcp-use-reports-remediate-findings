# Contributing Guidelines

Thank you for your interest in this Google Cloud security lab project!

## Goal of the Project

This repository demonstrates how to:
- Use Google Cloud Security Command Center (SCC)
- Analyze reports and findings
- Remediate misconfigurations and over-permissive IAM roles

Contributions are welcome if they:
- Improve explanations or documentation
- Add safer or clearer examples
- Fix typos, formatting, or structure
- Suggest better remediation or best practices

## What Not to Contribute

Please do **NOT** submit:
- Real organization IDs, project IDs, or resource names
- Sensitive screenshots with secrets or customer data
- Any proprietary information from production environments

Such content will be removed.

## How to Contribute

1. Fork this repository.
2. Create a new branch for your change.
3. Make your edits or improvements.
4. Open a Pull Request with:
   - A clear title
   - A short explanation of what you changed and why
