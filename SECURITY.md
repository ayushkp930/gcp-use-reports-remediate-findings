# Security Policy

## Supported Use

This project documents a **Google Cloud Security Command Center (SCC)** lab.
It is meant to show how to:

- Review security findings and misconfigurations
- Analyze IAM permissions
- Apply remediation steps to improve cloud security posture

It is **not** intended for attacking or abusing real-world environments.

## Reporting a Problem

If you notice:
- An unsafe example
- A missing warning
- Or something that could be misused

Please:

1. Avoid sharing sensitive details publicly.
2. Open a GitHub issue with a high-level description, **or**
3. Contact the author privately through GitHub.

## Legal & Ethical Requirements

- Only perform cloud security testing on projects or organizations where
  you have explicit authorization.
- Respect all local laws, company policies, and cloud provider terms of service.
- Do not use this project for malicious activities.
